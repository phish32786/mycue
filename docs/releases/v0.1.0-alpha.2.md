# MyCue 0.1.0-alpha.2

## Highlights

- Larger touch targets across the dashboard UI, including Spotify transport controls, launcher tiles, shared plugin actions, the page strip, and hardware overlay controls.
- Lower-right dashboard control HUD so `CTL`, `Edit`, and `Settings` no longer collide with the card editor controls on the right edge.
- Fixed hardware touch routing in layout edit mode so host controls like `Done` and `Edit` still respond while card outlines are visible.
- Added broader automated coverage for critical settings/model behavior and for F1 and launcher plugin logic.

## Included in this build

- Signed macOS app bundle
- Zip archive
- Drag-to-Applications DMG
